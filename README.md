CometSound
==========

Music player for GNU/Linux written in python, using pygtk and gstreamer libraries.

Copyright: 2010-2012 Andrea Bernardini

License: GPL-2+

Dependances:
	python2.6  
	python-gtk2  
	python-gst0.10  
	python-cerealizer  
	python-mutagen  
	python-notify  
	python-gobject  
	python-setproctitle  
	python-dbus  
	(gstreamer-plugins-bad for aac playback)
